\doxysubsection{Detailed Description}
Simple P\+ID controller. 

This P\+ID controller has a first-\/order filter on the derivative term, and windup limits. Since this is only used on our roll and pitch, this is sufficient. If to be used on other, less stable axes, filters should be reconsidered. Notably, if this needs to accomodate changes in setpoint, then the integral term needs to be fixed.
