\doxysubsection{Detailed Description}
The controls package contains code related to vehicle dynamics and motion controls. 

The primary controller implemented is 8D\+OF L\+QR with 2D\+OF P\+ID. We also implement fossen\textquotesingle{}s equations of motion for both simulation and optimal controls.
